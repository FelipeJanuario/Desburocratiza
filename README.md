# Desburocratiza
Esse projeto envolve a utilização de IA para geração de documentos jurídicos, sendo eles: 
- Petição
- Contestação
- Recurso
- Análises Jurídicas
- Gerador de Contratos
- Pesquisa de Leis
- Gerador de Cláusulas
- Gerador de Documentos e
- Buscador de Jurisprudência.
